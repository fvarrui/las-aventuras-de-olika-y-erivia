# Olika and Erivia Adventures

Platform game developed in [Godot Engine](https://godotengine.org/) based on a story written and drawn by my children.

![Screenshot](docs/screenshot.png)

## Controls

### Gamepad

- PS4 Dualshock:
  - Movement: **Left gear** 
  - Jump: **X**

### Keyboard

- Movement: **Cursors/Arrows**.
- Jump: **Space**.
- Fullscreen: **F**.
- Quit: **ESC**.

### Touchscreen
- On screen controls

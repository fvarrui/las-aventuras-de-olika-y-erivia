# Olika and Erivia Adventures

Platformer developed in [Godot Engine](https://godotengine.org/).

![Screenshot](docs/screenshot.png)

## Controls

### Gamepad

- PS4 Dualshock:
  - Movement: **Left gear** 
  - Jump: **X**

### Keyboard

- Movement: **Cursors/Arrows**.
- Jump: **Space**.
- Fullscreen: **F**.
- Quit: **ESC**.

### Touchscreen
- On screen controls
